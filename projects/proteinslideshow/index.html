Protein Explorer

Interactive viewer that spotlights residue neighborhoods, contact probabilities, and predicted side-chain conformations in a single pane.

0% Loading status…

Technical details

  • Frontend: vanilla HTML/CSS/JavaScript for maximum portability and minimal dependencies.
  • Responsive image grid with keyboard navigation and accessible controls.
  • Designed to load static image assets or integrate with simple JSON manifests for slide metadata (captions, citations, notes).
  • Optimized for reproducible export: image order and metadata can be serialized for reproducible figure generation.
