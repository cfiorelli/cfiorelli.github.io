Protein Explorer

Residue proximity explorer for PDB structures

  • protein-structure
  • visualization
  • ml

Interactive viewer that spotlights residue neighborhoods, contact probabilities, and predicted side-chain conformations in a single pane.

85% Soft launch

Deploying new contact maps to structural biology partners.

Technical details

  • Frontend: vanilla HTML/CSS/JavaScript for portability.
  • Residue neighborhood overlay renders from JSON manifolds exported by BioPython scripts.
  • Keyboard accessible navigation for stepping through structural slices.
  • Supports reproducible export of image order and captions for figure generation.
